# Assessment Preparation

## 5N1354 — Bookkeeping Manual and Computerised

**Assessment Split:** Project (50%) + Practical Exam (50%)

---

### Project (50%)

The project typically requires you to:

1. Process a set of transactions for a business through the complete bookkeeping cycle
2. Complete both manual and computerised versions
3. Reconcile results between the two
4. Prepare a VAT return
5. Print reports and submit backups

**Typical Project Tasks:**
- Enter source documents into daybooks (sales, purchases, cash, petty cash)
- Post to ledgers
- Extract trial balance
- Prepare bank reconciliation
- Calculate and prepare VAT return
- Complete the same cycle in accounting software
- Compare and reconcile manual vs computerised results
- Produce and print reports
- Backup data

**Time Allowed:** Usually 2-4 weeks

**Submission Includes:**
- Completed manual books (daybooks, ledgers, TB)
- Computerised report printouts
- Reconciliation notes
- Backup file (USB/DVD)

---

### Practical Exam (50%)

A supervised 2-hour exam where you process transactions in an accounts package.

**What to Expect:**
- Set up or open a company file
- Enter a series of transactions (10-15)
- Post invoices, receipts, payments
- Run a trial balance
- Generate a VAT return
- Print specific reports
- Backup data

**Marking Scheme (Indicative):**

| Area | Weight |
|------|--------|
| Data entry accuracy | 30% |
| Correct posting/ledgers | 25% |
| Trial Balance | 15% |
| VAT return accuracy | 15% |
| Reports and backup | 15% |

**Pass Mark:** 50% (with 50% minimum in each component)

### Sample Project Brief

**Business:** GreenGrocer Ltd — a fruit/veg wholesaler

**Given:**
- Opening trial balance
- 20 transactions (sales, purchases, returns, payments, receipts, petty cash)
- Bank statement extract

**Required:**
1. Complete all daybooks manually (with VAT analysis)
2. Post to ledgers
3. Extract trial balance
4. Prepare bank reconciliation
5. Calculate VAT return
6. Complete all steps in Sage/QuickBooks
7. Reconcile manual vs computerised results
8. Print TB, P&L, VAT report, Debtors/Creditors aging
9. Backup data to USB

## 5N1546 — Payroll Manual and Computerised

**Assessment Split:** Assignment (60%) + Exam (40%)

---

### Assignment (60%)

An extended project requiring comprehensive payroll processing.

**Typical Assignment Tasks:**
- Set up employee records with PPS numbers, tax credits, SRCOP
- Calculate gross pay including overtime, holiday pay, bonuses
- Process payroll under cumulative system for multiple employees
- Process emergency tax for one employee
- Handle mid-year starter/leaver
- Prepare P30, P45, P60
- Complete manual calculations
- Complete computerised calculations
- Compare and reconcile results

**Time Allowed:** 3-4 weeks

**Submission Includes:**
- Completed manual payroll calculations
- Computerised payroll reports
- Copies of P30, P45, P60
- Backup file

---

### Exam (40%)

A supervised 2-hour practical exam in payroll software.

**What to Expect:**
- Set up employees
- Process one or more payroll runs
- Apply different tax bases (cumulative/emergency)
- Generate payslips
- Prepare P30 summary
- Print reports and backup

**Marking Scheme (Indicative):**

| Area | Weight |
|------|--------|
| Employee setup & accuracy | 20% |
| Payroll calculation accuracy | 30% |
| Correct deductions (PAYE/PRSI/USC) | 25% |
| Revenue returns (P30) | 15% |
| Reports & backup | 10% |

**Pass Mark:** 50% (with 50% minimum in each component)

---

---

### Sample Assignment Brief

**Business:** TechSales Ltd — 4 employees

**Given:**
- Employee details (names, PPS, tax credits, SRCOP, pay rates)
- Time records for 4 weekly pay periods
- One employee started mid-period (emergency tax initially)
- One employee went on unpaid leave
- One employee received a bonus
- Overtime records

**Required:**
1. Calculate gross pay for each employee for all 4 weeks (manual)
2. Calculate PAYE, PRSI, USC manually using cumulative system
3. Calculate net pay
4. Process Week 1 employee under emergency tax → switch to cumulative in Week 3
5. Generate P45 for leaver (Week 4)
6. Complete all steps in payroll software
7. Compare manual vs computerised results
8. Generate P30 for the month
9. Print payslips and payroll summary
10. Backup data

## General Assessment Tips

1. **Manual First** — Always do manual calculations first before software. This ensures you understand the principles.
2. **Check Your Work** — Trial balance, control accounts, and bank reconciliation are all verification tools.
3. **Reconcile** — Manual and computerised results must match. If they don't, find out why.
4. **Backup Everything** — Both for the assessment and real-world practice.
5. **Read the Question** — In exams, read every word. Many marks are lost for missing specific requirements.
6. **Time Management** — Allocate time per task. Don't spend too long on one section.
