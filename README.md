# Bookkeeping & Payroll Course

## QQI Level 5 — 5N1354 + 5N1546

Two 15-credit QQI minor awards designed for you to complete together.

| Code | Title | Credits | Level |
|------|-------|---------|-------|
| 5N1354 | Bookkeeping Manual and Computerised | 15 | 5 |
| 5N1546 | Payroll Manual and Computerised | 15 | 5 |

### Structure

```
bookkeeping/       → 5N1354 (10 learning outcomes, 10 lessons)
payroll/           → 5N1546 (10 learning outcomes, 10 lessons)
exercises/         → Practical worksheets for both modules
assessment/        → Exam prep, sample papers, project guides
```

### Prerequisites

- NFQ Level 4 or equivalent
- Basic computer literacy (spreadsheets, file management)
- English proficiency (CEFR B2+)

### Recommended Software

- Sage 50 / QuickBooks / Tally or similar accounting package
- Thesaurus Payroll / Sage Payroll or similar payroll software
- Microsoft Excel / Google Sheets
- Revenue Online Service (ROS) familiarity

### Assessment

**5N1354 Bookkeeping**: Project (50%) + Practical Exam (50%)
**5N1546 Payroll**: Assignment (60%) + Exam (40%)

Grading: Pass 50-64% | Merit 65-79% | Distinction 80-100%

### Full Course Contents

```
bookkeeping/           → 5N1354 — 10 lessons with worked examples
payroll/               → 5N1546 — 10 lessons with worked examples
exercises/             → Practical worksheets for both modules
exercise-solutions/    → Full worked solutions with step-by-step calculations
templates/             → Blank forms: daybooks, ledgers, TB, cash book,
                         petty cash, bank reconciliation, VAT return,
                         payroll calculation sheet, P30, P45
mock-projects/         → Full mock QQI assessments with solutions
software-guides/       → Step-by-step for Sage/QuickBooks and
                         Thesaurus/Sage Payroll
reference/             → Glossary of 70+ terms + cheat sheet
assessment/            → Assessment prep, marking schemes, study plans
```

### How to Use This Course

1. Work through each outcome in order within a module
2. Complete the exercises after each topic — check your answers against the solutions
3. Use the blank templates for manual practice
4. Reconcile manual vs computerised results at each step
5. Complete the mock projects under timed conditions
6. Use the reference materials for quick lookup during study
7. Review the assessment prep for exam technique
